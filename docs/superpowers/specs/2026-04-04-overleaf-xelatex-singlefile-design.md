标题：生成新版 Overleaf XeLaTeX 单文件设计（方案一）
日期：2026-04-04

目标

基于最新 `本科毕业论文-研究报告.md`，生成新的单文件 XeLaTeX 代码（`main.tex` 风格），并放置在 workspace 根目录。该版本需兼顾：

1. Overleaf 可直接编译（XeLaTeX）
2. 版式规范，适合本科论文阅读
3. 导出到 WPS PDF 后页数控制在十几页到二十页范围（目标 15-20 页）

范围

- 输入：`本科毕业论文-研究报告.md`
- 输出：根目录固定新文件 `main_xelatex_rigorous.tex`（单文件、可直接编译）
- 不修改论文核心内容与实验结论，仅做 LaTeX 结构与排版控制

结构映射规则（必须固定）

- 文档层级：采用 `article` 体例，映射为 `\section` / `\subsection`。
- 标题与摘要顺序：题目 -> 中文摘要 -> 中文关键词 -> 英文摘要 -> 英文关键词 -> 目录 -> 正文 -> 参考文献 -> 附录。
- Markdown 原有“第1章/1.1”文本保留在标题内容中，不自动改写为 `\chapter`。

参考文献规范

- 来源：沿用 Markdown 中已有手写参考文献条目（不引入 BibTeX）。
- 格式：使用 `thebibliography` 环境或等效编号列表，保证编号连续、可编译。
- 文中引用保留现状（如 `[1]`）并保持与参考文献编号一致。

引用校验规则

- 校验文中出现的编号集合与参考文献条目编号集合一致。
- 若发现编号缺失或错位：以参考文献条目顺序为准修正文中编号，并记录修复说明。

执行方案

1. 自动转换
   - 使用 Pandoc 将 Markdown 转换为 LaTeX 正文草稿。

2. 手工排版增强（单文件）
   - 补齐文档类与核心宏包：`ctex`/`xeCJK`、`geometry`、`setspace`、`graphicx`、`booktabs`、`longtable`、`caption`、`hyperref`。
   - 设置页面与字号：A4、12pt、页边距约 2.5cm。
   - 设置行距与段落：行距 1.3 左右，适度段间距。
   - 增加目录与目录页：保证论文可读性。
   - 统一图表标题格式与编号显示。

3. 页数控制策略（15-20 页）
   - 通过行距、图宽和浮动体位置控制页数：
     - 图默认 `width=0.82\textwidth`（必要时 0.78~0.88 微调）
     - 避免单图占整页
   - 合理插入分页断点（摘要后、正文开始前、参考文献前）。
- 保持表格紧凑但可读（适度 `\arraystretch`）。

页数调参优先级（严格执行）

1. 优先调图宽（0.82 -> 0.78 或 0.86）
2. 再调行距（1.30 -> 1.25 或 1.35）
3. 最后微调分页断点
4. 不允许通过删除正文段落达成页数目标

4. 兼容性说明
- 以 Overleaf XeLaTeX 为第一目标；字体使用 Overleaf 常见可用字体（如 Fandol/TeX Gyre 组合）。
- 保留图片相对路径，假设图片与 `main_xelatex_rigorous.tex` 同目录。

编译与 fallback 约定

- 编译器：Overleaf `XeLaTeX`（latexmk 默认流程）。
- 字体 fallback：提供两套可切换预设（默认 Fandol，备选 Noto CJK），通过导言区注释开关手动切换，不做条件自动探测。
- 图片缺失处理：保留 `\includegraphics` 语句并在注释中提示缺失文件名，不静默删除图引用。

编译验收步骤（必须执行）

1. 使用 XeLaTeX 编译至少 1 次（本地或 Overleaf 等价流程）。
2. 记录“无致命错误”结果（允许非阻断警告）。
3. 若首次失败，按日志修复语法/宏包问题后再次编译。

验收标准

- 生成新文件 `main_xelatex_rigorous.tex` 位于 workspace 根目录，且为单文件 LaTeX 完整代码。
- 代码包含 `\documentclass ... \begin{document} ... \end{document}`。
- 包含目录、标题、正文、参考文献段落。
- 图表、公式、表格可正常显示（语法层面）。
- 页数控制参数已明确配置，目标区间 15-20 页（WPS 导出时可微调图宽/行距）。

页数判定口径与停止条件

- 终判口径：以 WPS 导出 PDF 页数为准。
- 目标区间：15-20 页。
- 若超出区间：按“图宽 -> 行距 -> 分页断点”顺序最多调参两轮；两轮后仍超出则保留可读性优先并在交付说明中注明。
